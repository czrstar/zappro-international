FAQ

Common questions

Does this work with WhatsApp Cloud API?

The Core template uses Evolution API (connects to your WhatsApp Business App via QR code). The Pro tier includes a migration guide for WhatsApp Cloud API + Coexistence, which lets you use both the app and the bot simultaneously.

Which AI models are supported?

Any model available on OpenRouter — Claude 3.5 Haiku, GPT-4o, Gemini 2.0 Flash, Llama 3, and hundreds more. The fallback uses Anthropic's API directly. You can swap models by changing one field in n8n.

Do I need a server to run this?

Yes — n8n must be self-hosted (a $6–12/month VPS works fine). n8n Cloud also works. The PostgreSQL database can run on the same server or on a managed service like Supabase or Railway.

Can I use this for multiple clients?

Yes. Each client gets their own duplicated workflow with a unique webhook path and Evolution API instance. The Pro tier includes a guide on running 10+ clients on a single n8n instance without interference.
